# 教案录入案例

## PilotDeck Prompt

```text
用 $local-document-entry 处理这张教案图片：
C:\Users\a1866\.codex\attachments\6da70f50-38b0-4aa8-abaf-ab305f7b3fa0\image-1.png

字段：
授课日期、授课班级、授课题目、所用教材、任课教师

要求：
1. 使用本地模式。
2. 导出 Excel。
3. 生成 Markdown 复核报告。
4. 告诉我低置信度字段和缺失字段。
```

## 预期输出

- 一个任务 ID。
- 一个 Excel 文件，按字段列导出，每页一行。
- 一个 Markdown 报告，包含处理链路、页数、低置信度字段、缺失字段和导出路径。

## 演示重点

- PilotDeck 不是模型本身，而是调度层。
- 文档工作台是本地执行引擎。
- MiniCPM5-1B 不直接“硬读图”，只在 OCR/VL、候选召回和结构判断之后做字段裁决。
